Football Stats Project

Project Overview

Welcome to the Football Stats Project! This project is designed to provide an interactive platform to analyze and compare football statistics, allowing you to explore detailed team performance, player efficiency, and compare different leagues.

The project leverages data scraping techniques to extract real-time data from football statistics websites and displays this data in a user-friendly format. You can view team performance, player efficiency, and compare the top players across different leagues.

Functionalities

  • Team Performance: Explore detailed performance statistics for your favorite football teams. This includes goals, attacking, possession, defending, and more.
  • Player Efficiency: Compare players based on their shooting efficiency (goals per shots on target). This feature allows you to see which players are the most accurate in front of goal.
  • Compare Leagues: Compare the top players from different leagues based on their shooting precision. This visual comparison helps in identifying the best shooters across different leagues.

How It Works

This project uses a combination of web scraping with BeautifulSoup and Selenium to gather the most up-to-date statistics from trusted football websites. The data is then processed and visualized using Chart.js to create interactive charts.
